Mixed Languages/Protocols

Another confusing stuff is that there are more than one languages or protocols mixed in one document.

For instance, JSX, ES6, Java Script, HTML, CSS and so on.

You have to carefully tell one from others, otherwise you may confused and apply wrong syntax.

So learn a bit more how function/or method, class, object are defined in ES6, in Java Script is necessary.

Then you should have a clear sight about the code itself.

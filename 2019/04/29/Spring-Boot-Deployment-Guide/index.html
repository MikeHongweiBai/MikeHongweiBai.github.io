Spring and Spring Boot

A number of people on the Internet compare them as Servlet and Spring MVC.

Spring Boot is based on Spring and it is much easier to config and deploy.

While Some of them complains about Spring Boot does not have xml config, while they config in java.

Spring Boot is more suitable for small and fast project.

If you want to select a framework for a bigger project or an organization, go for Spring Cloud, which is based on Spring Boot and has config files.

Anyway, I choose to study Spring Boot for now.

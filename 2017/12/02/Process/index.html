Process

2017-12-02
#Process
Android/ Java/ Linux

进程(Process)是系统资源分配和调度的基本单位。
Process is a basic unit for system to allocate and dispatch.

一般情况下，进程有三种状态：
There are three states for a process in general:

就绪，运行和阻塞。
Ready, Run and Blocked

  • Run: process occupies CPU and is running on CPU.
    一个进程如果占用CPU时间过长，系统分配给每个进程的时间片是有限的。这里牵扯出另一个概念：即操作系统的CPU调度算法，例如：抢先式优先级调度算法等。
  • Ready: process is ready for running, but CPU has not been allocated to this process.
  • Blocked: process could not run because the process is waiting for something to happen.

linkto: 操作系统调度

三种状态的转换：
[Run] —- I/O request —-> [Blocked] —- I/O finished —-> [Ready]
[Ready] —-Dispatch —-> [Run]
[Run] —- Time slot finished —-> [Ready]
有两种转换是不可能出现的：

  • 阻塞 -> 运行
  • 就绪 -> 阻塞
